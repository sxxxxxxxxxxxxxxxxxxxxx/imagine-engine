## 🎯 使用流程

### 首次使用（5分钟）
```
1. 打开应用
   http://localhost:3000

2. 配置API（重要！）
   点击 ⚙️ 设置
   选择 Nano Banana
   输入 API密钥
   保存设置

3. 开始创作
   按 G 键 → AI创作
   按 P 键 → 打开画廊
   选择模板 → 生成

4. 探索功能
   按 ? → 查看所有快捷键
   按 L → 浏览案例
   按 E → 尝试编辑
```
